## 📚 Configuration

### 📕 .prettierrc

```json
{
  "singleQuote": true, 
  "semi": true,
  "useTabs": false,
  "tabWidth": 2,
  "trailingComma": "all",
  "printWidth": 80,
  "arrowParens": "always",
  "orderedImports": true
}
```
|설정|내용|
|---|---|
|singleQuote|single 쿼테이션 이용 여부|
|semi|세미클론 이용|
|useTabs|탭 사용 여부|
|tabWidth|탭 너비|
|trailingComma|여러 줄 사용시, 후행 콤마 사용 방식|
|printWidth|줄 바꿈 할 폭 길이|
|arrowParens|화살표 함수 괄호 표기 방식|
|orderedImports|import 정렬 여부|
